wage_data=visa_df['prevailing_wage'] # take one num col which is having outliers

q1=round(np.quantile(wage_data,0.25),2)  # calculate Q1 (25 percentile)
q2=round(np.quantile(wage_data,0.50),2)  # calculate Q2 (50 percentile)
q3=round(np.quantile(wage_data,0.75),2)  # calculate Q3 (75 percentile)

iqr=q3-q1  # calculate IQR

lb=q1-1.5*iqr  # Lower bound
ub=q3+1.5*iqr # upper bound

con1=wage_data<lb   # condition 1
con2=wage_data>ub   # condition 2

con3=con1|con2    # condition 3
outliers_data=visa_df[con3]   # True False value converted into data frame

How to deal the outliers

Method-1: Drop the outliers

  • if outliers has less than 2% of total data, we can drop the outliers

  • drop the outliers is not a recommended approch

  • if we drop the outliers other values also drop

  • we encounter a situation about to loss the data

  • in above we developed a code for non-outliers data

Method-2: Fill with Median

  • we already know that outliers does not affect by median

  • so it is a good approch fill the outliers with median

Method-3: Cap the outliers

  • instead of fill the outliers with median, we can fill with Q3 or Q1 or max or min value

In[258]:
wage_data=visa_df['prevailing_wage']

q1=round(np.quantile(wage_data,0.25),2) 
q2=round(np.quantile(wage_data,0.50),2)
q3=round(np.quantile(wage_data,0.75),2)

iqr=q3-q1

lb=q1-1.5*iqr
ub=q3+1.5*iqr

median=wage_data.median()

new_data=[]
for val in wage_data:
    if val<lb or val>ub:
        new_data.append(median)

    else:
        new_data.append(val)
        
visa_df['prevailing_wage']=new_data

instead of 'for loop' and 'if condition' we use 'np.where'

np.where

  • np.where is a method useful when we can fill the values with one condition two outputs

  • all the if-else statements we can do with np.where

  • syntax: np.where(condition,True value,False value)

  • it takes 3 arguments

    • Condition

    • True value: when the condition becomes True

    • False value: when the condition becomes False

ub=q3+1.5*iqr

con1=wage_data<lb
